Real-Time Human Tracking with Lidar and Camera Fusion

This project demonstrates a multi-sensor fusion approach for robust real-time human tracking, utilizing both Lidar and camera data.

This human-tracking system combines Lidar and camera data for accurate position estimation, even in cluttered environments. The project addresses challenges in sensor synchronization, occlusion management, and false-positive reduction using predictive modeling and data fusion algorithms.

  • View Code on GitHub

Calibration and Data Synchronization

Data fusion begins with intrinsic and extrinsic calibration to align Lidar and camera frames. Synchronization is managed using a ROS-based time synchronizer, ensuring sensor data captured within a 0.6-second window are processed together for real-time performance.

Fusion and Detection Techniques

The system leverages YOLO-based camera detections to filter false positives from Lidar detections, using projection techniques and the Hungarian algorithm for data association. By projecting Lidar points onto the camera frame, reliable human tracking is achieved.

Challenges and Occlusion Management

Handling occlusions in crowded environments is crucial. The Extended Kalman Filter (EKF) enables predictions when measurements are unavailable, ensuring consistent tracking of individuals even during temporary occlusions.
