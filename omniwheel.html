Omni-Wheeled Drawing Robot

This project showcases my work on an omni-wheeled robot that can draw intricate patterns on any surface.

The omni-wheeled robot was designed to handle multi-directional movement, enabling it to draw complex designs. The project involved integrating motors, sensors, and a control system, providing precision and agility in robot movement.

  • View Code on GitHub

Challenges and Solutions

One of the biggest challenges was achieving smooth, precise control over the robot's movements. I implemented PID control to fine-tune the speed and direction of the wheels, ensuring accurate drawings.

Technologies Used

This project utilizes various technologies, including Arduino for microcontroller programming, ROS for robot communication, and Python for control algorithms.
